1. 輾轉相除法 國中

想找兩個數的最大公因數？不斷用小的去除大的，直到餘數是 0，最後那個除數就是答案！這招叫做「輾轉相除法」（歐幾里得算法），已經有 2300 年歷史了。

🔍 看看怎麼運作

求 gcd(48, 18)：「不斷用小的去除大的，直到餘數是 0」

⚡ 哪個方法快？

找 gcd(252, 105)：

列出因數法

252 的因數：1,2,3,4,6,7,9,12,14,18,21,28,36,42,63,84,126,252

105 的因數：1,3,5,7,15,21,35,105

要列 26 個數

輾轉相除法

252÷105=2 餘 42

105÷42=2 餘 21

42÷21=2 餘 0 → 答案 21

只要 3 步！

數字越大差距越明顯。gcd(10946, 6765) 列因數要花超久，但輾轉相除法也只要幾步！

2. 同餘與模運算 國中/高中

13 點其實就是下午 1 點——因為時鐘只有 12 格！這就是「模運算」：只看除法的餘數。數學裡寫成 a ≡ b (mod n)。

🕐 時鐘類比

a ≡ b (mod n) 意思是「a 和 b 除以 n 的餘數一樣」

例如：13 ≡ 1 (mod 12)，因為 13 ÷ 12 = 1 餘 1

又如：25 ≡ 1 (mod 12)，因為 25 ÷ 12 = 2 餘 1

✨ 模運算的超能力

加法性質：

(a + b) mod n = ((a mod n) + (b mod n)) mod n

例：(17+23) mod 5 = (2+3) mod 5 = 5 mod 5 = 0 ✓

乘法性質：

(a × b) mod n = ((a mod n) × (b mod n)) mod n

例：(17×23) mod 5 = (2×3) mod 5 = 6 mod 5 = 1 ✓

為什麼有用？

計算 99^99 mod 7：直接算 99^99 是天文數字，但 99 mod 7 = 1，所以 99^99 mod 7 = 1^99 mod 7 = 1！

3. 費馬小定理 & 歐拉定理 高中

如果 p 是質數，而 a 不是 p 的倍數，那麼 a^(p-1) ≡ 1 (mod p)。這是 1640 年費馬發現的魔法定理！

🔮 費馬小定理的魔法

觀察 2 的各種次方 mod 7：

2¹ mod 7 = 2
2² mod 7 = 4
2³ mod 7 = 8 mod 7 = 1 ← 哇？
2⁴ mod 7 = 16 mod 7 = 2 ← 又回來了！
2⁵ mod 7 = 32 mod 7 = 4
2⁶ mod 7 = 64 mod 7 = 1 ← 費馬保證 p-1=6 次方一定回到 1！

定理：若 p 是質數且 gcd(a,p) = 1，則 a^(p-1) ≡ 1 (mod p)

🌟 歐拉定理（費馬的推廣）

歐拉定理：若 gcd(a,n) = 1，則 a^φ(n) ≡ 1 (mod n)

φ(n)（歐拉函數）= 小於 n 且與 n 互質的正整數個數

如何計算 φ(n)：

先做質因數分解 n = p₁^a₁ × p₂^a₂ × ...

然後 φ(n) = n × (1-1/p₁) × (1-1/p₂) × ...

範例：φ(12) = ?

12 = 2² × 3

φ(12) = 12 × (1-1/2) × (1-1/3) = 12 × 1/2 × 2/3 = 4

驗證：與 12 互質的數 = {1, 5, 7, 11}，共 4 個 ✓

4. 中國剩餘定理 高中

古代的「韓信點兵」問題——士兵排 3 人一列餘 2，排 5 人一列餘 3，排 7 人一列餘 2，問最少幾人？這個 1800 年前的定理至今仍是密碼學的基石！

📜 韓信點兵的故事

數學語言：找最小正整數 x 使得

x ≡ 2 (mod 3)
x ≡ 3 (mod 5)
x ≡ 2 (mod 7)

解題思路：列出每個條件的候選人，找交集！

x mod 3 = 2 → 2, 5, 8, 11, 14, 17, 20, 23, 26, 29...
x mod 5 = 3 → 3, 8, 13, 18, 23, 28, 33, 38...
x mod 7 = 2 → 2, 9, 16, 23, 30, 37, 44...
→ 三條線交在 x = 23！

驗算：23÷3=7 餘 2 ✓ | 23÷5=4 餘 3 ✓ | 23÷7=3 餘 2 ✓

所有解：23, 23+105, 23+210, ... （解之間差 3×5×7=105）

🔧 建構式解法

N = 3×5×7 = 105

N₁ = 105/3 = 35, 找 35t₁ ≡ 1 (mod 3) → t₁ = 2

N₂ = 105/5 = 21, 找 21t₂ ≡ 1 (mod 5) → t₂ = 1

N₃ = 105/7 = 15, 找 15t₃ ≡ 1 (mod 7) → t₃ = 1

x = 2×35×2 + 3×21×1 + 2×15×1 = 140+63+30 = 233 ≡ 23 (mod 105)

6. 完全數與特殊數 國中/高中

古希臘人認為某些數字擁有「完美」的性質：它所有因數（不含自己）加起來，剛好等於自己！2500 年後，我們仍然不知道有沒有「奇數完全數」存在。

✨ 完全數 (Perfect Numbers)

定義：σ(n) - n = n，即真因數之和等於自己

6 的真因數：1, 2, 3 → 1+2+3 = 6 → 完全！

28 的真因數：1, 2, 4, 7, 14 → 1+2+4+7+14 = 28 → 完全！

496 的真因數之和 = 496 → 第三個完全數！

過剩數 (Abundant)：因數之和 > 自己

12：1+2+3+4+6 = 16 > 12

不足數 (Deficient)：因數之和 < 自己

8：1+2+4 = 7 < 8

💕 友誼數 (Amicable Numbers)

220 的真因數之和 = 284

284 的真因數之和 = 220

它們互相指向對方，像一對好朋友！

其他友誼數對：(1184, 1210), (2620, 2924), (5020, 5564)...

🌊 梅森質數 (Mersenne Primes)

形式：M_p = 2^p - 1（p 本身必須是質數）

M₂ = 2²-1 = 3 ← 質數 ✓
M₃ = 2³-1 = 7 ← 質數 ✓
M₅ = 2⁵-1 = 31 ← 質數 ✓
M₇ = 2⁷-1 = 127 ← 質數 ✓
M₁₁ = 2¹¹-1 = 2047 = 23×89 ← 不是質數 ✗
M₁₃ = 2¹³-1 = 8191 ← 質數 ✓

🏆 目前最大已知質數是 2^136279841 - 1，有超過 4100 萬位數！由 GIMPS 分散式計算計畫在 2024 年發現。

💡 每個偶完全數都可以寫成 2^(p-1) × (2^p - 1)，其中 2^p-1 是梅森質數。

7. 二次剩餘與原根 大學

在 mod p 的世界裡，有些數可以被「平方」得到，有些不行。而「原根」是一個超級基底——它的各種次方能產生所有非零元素！

🔢 二次剩餘 (Quadratic Residues)

問題：x² ≡ a (mod p) 有沒有整數解 x？

以 mod 7 為例，計算所有平方：

1² mod 7 = 1 ✓
2² mod 7 = 4 ✓
3² mod 7 = 9 mod 7 = 2 ✓
4² mod 7 = 16 mod 7 = 2（重複）
5² mod 7 = 25 mod 7 = 4（重複）
6² mod 7 = 36 mod 7 = 1（重複）

QR mod 7 = {1, 2, 4} NQR = {3, 5, 6}

x² ≡ 3 (mod 7) 無解！因為 3 不在 QR 裡面。

Legendre 符號：(a/p) = 1 若 a 是 QR，= -1 若是 NQR，= 0 若 p|a

快速判斷：(a/p) ≡ a^((p-1)/2) (mod p)（歐拉判準）

⚡ 原根 (Primitive Root)

定義：g 是 mod p 的原根 ⟺ {g¹, g², ..., g^(p-1)} mod p = {1, 2, ..., p-1}

即 g 的次方能「生出」所有非零元素！

範例：3 是 mod 7 的原根嗎？

3¹ mod 7 = 3
3² mod 7 = 9 mod 7 = 2
3³ mod 7 = 27 mod 7 = 6
3⁴ mod 7 = 81 mod 7 = 4
3⁵ mod 7 = 243 mod 7 = 5
3⁶ mod 7 = 729 mod 7 = 1
→ 生出了 {3,2,6,4,5,1} = {1,2,3,4,5,6} 全部！是原根！

2 是 mod 7 的原根嗎？

2¹=2, 2²=4, 2³=1, 2⁴=2, 2⁵=4, 2⁶=1
只生出 {1,2,4}，週期是 3 不是 6。不是原根！

每個質數 p 都有原根。原根的個數 = φ(p-1)。

8. Diophantine 方程 大學

只要求「整數解」的方程式，以古希臘數學家丟番圖命名。從畢達哥拉斯的直角三角形到費馬最後定理，這些問題困擾了數學家幾千年！

📐 線性 Diophantine 方程

問題：ax + by = c，何時有整數解 (x, y)？

定理：有整數解 ⟺ gcd(a,b) | c

例 1：3x + 5y = 1

gcd(3,5) = 1，而 1|1 → 有解！

一組解：x=2, y=-1（驗算：6-5=1 ✓）

例 2：6x + 4y = 3

gcd(6,4) = 2，而 2∤3 → 無解！

（左邊永遠是偶數，不可能等於奇數 3）

🔄 擴展歐幾里得算法

不只能找 gcd，還能找出 ax + by = gcd(a,b) 的解！

範例：找 x, y 使得 35x + 15y = gcd(35,15) = 5

📐 畢氏三元組 (Pythagorean Triples)

問題：找所有正整數 (a, b, c) 使得 a² + b² = c²

公式：所有原始畢氏三元組可以寫成：

a = m²-n², b = 2mn, c = m²+n²

（m > n > 0，m 和 n 互質且奇偶不同）

m=2, n=1 → a=3, b=4, c=5 | 9+16=25 ✓
m=3, n=2 → a=5, b=12, c=13 | 25+144=169 ✓
m=4, n=1 → a=15, b=8, c=17 | 225+64=289 ✓
m=4, n=3 → a=7, b=24, c=25 | 49+576=625 ✓
m=5, n=2 → a=21, b=20, c=29 | 441+400=841 ✓
m=5, n=4 → a=9, b=40, c=41 | 81+1600=1681 ✓

💡 費馬最後定理：a^n + b^n = c^n 在 n≥3 時沒有正整數解！（1995 年 Andrew Wiles 證明）

🌀 Pell 方程

問題：x² - Dy² = 1（D 不是完全平方數）

以 D=2 為例：x² - 2y² = 1

(x, y) = (1, 0) — 平凡解
(x, y) = (3, 2) — 9 - 8 = 1 ✓
(x, y) = (17, 12) — 289 - 288 = 1 ✓
(x, y) = (99, 70) — 9801 - 9800 = 1 ✓
(x, y) = (577, 408) — 332929 - 332928 = 1 ✓

遞推公式：x_{n+1} = 3x_n + 4y_n，y_{n+1} = 2x_n + 3y_n

Pell 方程保證有無限多組解！最小正解可以用連分數展開 √D 來找。
