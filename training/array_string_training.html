Technique Selection Guide

Array and String problems form ~40% of coding interviews. The key is quickly recognizing which technique applies.

Decision Framework

Complexity Quick Reference

n = 10^5: O(n log n) is safe, O(n²) will TLE
n = 10^4: O(n²) borderline, O(n² log n) TLE
n = 10^3: O(n²) safe, O(n³) borderline
n = 20: O(2^n) or O(n * 2^n) fine

Two Pointers

Use two indices to traverse the array in a coordinated way. Works best on sorted arrays or when comparing elements from both ends.

Pattern 1: Opposite Direction (left + right)

// LC 167 Two Sum II (sorted array)
int left = 0, right = n - 1;
while (left < right) {
    int sum = nums[left] + nums[right];
    if (sum == target) return new int[]{left, right};
    else if (sum < target) left++;
    else right--;
}

Pattern 2: Same Direction (fast + slow)

// LC 26 Remove Duplicates from Sorted Array
int slow = 0;
for (int fast = 1; fast < n; fast++) {
    if (nums[fast] != nums[slow]) {
        slow++;
        nums[slow] = nums[fast];
    }
}
return slow + 1;

Pattern 3: Three Pointers (3Sum)

// LC 15 3Sum: sort + fix one + two pointers
Arrays.sort(nums);
for (int i = 0; i < n - 2; i++) {
    if (i > 0 && nums[i] == nums[i-1]) continue;  // skip dup
    int lo = i + 1, hi = n - 1;
    while (lo < hi) {
        int sum = nums[i] + nums[lo] + nums[hi];
        if (sum == 0) {
            result.add(Arrays.asList(nums[i], nums[lo], nums[hi]));
            while (lo < hi && nums[lo] == nums[lo+1]) lo++;
            while (lo < hi && nums[hi] == nums[hi-1]) hi--;
            lo++; hi--;
        } else if (sum < 0) lo++;
        else hi--;
    }
}

Pattern 4: Partition (Dutch National Flag)

// LC 75 Sort Colors: 0s left, 2s right, 1s middle
int lo = 0, mid = 0, hi = n - 1;
while (mid <= hi) {
    if (nums[mid] == 0) swap(nums, lo++, mid++);
    else if (nums[mid] == 2) swap(nums, mid, hi--);
    else mid++;
}

Sliding Window

Maintain a window [left, right] that expands right and shrinks left. O(n) for "longest/shortest subarray/substring with property X".

Template: Variable-Size Window (Longest)

int left = 0, maxLen = 0;
// state: e.g., HashMap for char frequency

for (int right = 0; right < n; right++) {
    // 1. Expand: add nums[right] to window state
    addToWindow(nums[right]);

    // 2. Shrink: while window is invalid
    while (!isValid()) {
        removeFromWindow(nums[left]);
        left++;
    }

    // 3. Update answer
    maxLen = Math.max(maxLen, right - left + 1);
}

Template: Variable-Size Window (Shortest)

int left = 0, minLen = Integer.MAX_VALUE;

for (int right = 0; right < n; right++) {
    addToWindow(nums[right]);

    // Shrink while window IS valid (find minimum)
    while (isValid()) {
        minLen = Math.min(minLen, right - left + 1);
        removeFromWindow(nums[left]);
        left++;
    }
}

Template: Fixed-Size Window

for (int right = 0; right < n; right++) {
    addToWindow(nums[right]);
    if (right >= k - 1) {
        // window [right-k+1, right] is full
        updateAnswer();
        removeFromWindow(nums[right - k + 1]);
    }
}

When to Use Which

"Longest substring without repeating" → expand right, shrink left when invalid
"Minimum window substring" → expand right, shrink left while valid
"Max sum of subarray size k" → fixed window
"At most K distinct" → variable, shrink when distinct > K

Classic: Minimum Window Substring

// LC 76: find shortest substring of s containing all chars of t
int[] need = new int[128], have = new int[128];
for (char c : t.toCharArray()) need[c]++;
int required = t.length(), formed = 0;
int left = 0, minLen = Integer.MAX_VALUE, start = 0;

for (int right = 0; right < s.length(); right++) {
    char c = s.charAt(right);
    have[c]++;
    if (have[c] <= need[c]) formed++;

    while (formed == required) {
        if (right - left + 1 < minLen) {
            minLen = right - left + 1;
            start = left;
        }
        char d = s.charAt(left);
        have[d]--;
        if (have[d] < need[d]) formed--;
        left++;
    }
}

Prefix Sum

Precompute cumulative sums to answer range sum queries in O(1). Combines powerfully with HashMap for "subarray sum = k" problems.

Basic Template

// prefix[i] = sum of nums[0..i-1]
int[] prefix = new int[n + 1];
for (int i = 0; i < n; i++)
    prefix[i + 1] = prefix[i] + nums[i];

// Sum of nums[l..r] = prefix[r+1] - prefix[l]

Pattern: Subarray Sum Equals K (Prefix + HashMap)

// LC 560: count subarrays with sum = k
Map<Integer, Integer> map = new HashMap<>();
map.put(0, 1);  // empty prefix
int sum = 0, count = 0;

for (int num : nums) {
    sum += num;
    count += map.getOrDefault(sum - k, 0);
    map.merge(sum, 1, Integer::sum);
}

2D Prefix Sum

// LC 304 Range Sum Query 2D
// prefix[i][j] = sum of rectangle (0,0) to (i-1,j-1)
for (int i = 1; i <= m; i++)
    for (int j = 1; j <= n; j++)
        prefix[i][j] = matrix[i-1][j-1]
            + prefix[i-1][j] + prefix[i][j-1] - prefix[i-1][j-1];

// Sum of (r1,c1) to (r2,c2):
// prefix[r2+1][c2+1] - prefix[r1][c2+1] - prefix[r2+1][c1] + prefix[r1][c1]

Difference Array (range update in O(1))

// Add val to range [l, r]:
diff[l] += val;
diff[r + 1] -= val;
// Reconstruct: prefix sum of diff gives actual array

HashMap / HashSet Techniques

Trade space for time. O(1) lookup transforms O(n²) brute force into O(n).

Pattern 1: Two Sum (complement lookup)

// LC 1 Two Sum
Map<Integer, Integer> map = new HashMap<>();
for (int i = 0; i < n; i++) {
    int complement = target - nums[i];
    if (map.containsKey(complement))
        return new int[]{map.get(complement), i};
    map.put(nums[i], i);
}

Pattern 2: Frequency Count

// LC 347 Top K Frequent Elements
Map<Integer, Integer> freq = new HashMap<>();
for (int num : nums) freq.merge(num, 1, Integer::sum);
// Then use bucket sort or heap to get top k

// LC 49 Group Anagrams: sort each word as key
Map<String, List<String>> map = new HashMap<>();
for (String s : strs) {
    char[] chars = s.toCharArray();
    Arrays.sort(chars);
    String key = new String(chars);
    map.computeIfAbsent(key, k -> new ArrayList<>()).add(s);
}

Sorting Tricks

Sort to impose structure, then apply linear scan or binary search.

Custom Comparators

// LC 179 Largest Number: custom string comparison
String[] strs = new String[nums.length];
for (int i = 0; i < nums.length; i++) strs[i] = String.valueOf(nums[i]);
Arrays.sort(strs, (a, b) -> (b + a).compareTo(a + b));

Bucket Sort: O(n) when range is bounded

// LC 347 Top K Frequent: bucket[freq] = list of elements with that freq
List<Integer>[] buckets = new List[n + 1];
for (Map.Entry<Integer, Integer> e : freq.entrySet()) {
    int f = e.getValue();
    if (buckets[f] == null) buckets[f] = new ArrayList<>();
    buckets[f].add(e.getKey());
}

Cyclic Sort: Find missing/duplicate in [1, n]

// LC 268 Missing Number / LC 442 Find Duplicates
for (int i = 0; i < n; i++) {
    while (nums[i] > 0 && nums[i] <= n && nums[nums[i] - 1] != nums[i])
        swap(nums, i, nums[i] - 1);
}
// After: nums[i] should be i+1. Anything else = missing/dup

Binary Search on Arrays

Not just for sorted arrays! Use binary search on the answer space when the predicate is monotone.

Template: Find Exact / Lower Bound / Upper Bound

// Lower bound: first index where nums[i] >= target
int lo = 0, hi = n;
while (lo < hi) {
    int mid = lo + (hi - lo) / 2;
    if (nums[mid] < target) lo = mid + 1;
    else hi = mid;
}
// lo = insertion point (first >= target)

Template: Binary Search on Answer

// "Find minimum X such that condition(X) is true"
int lo = minPossible, hi = maxPossible;
while (lo < hi) {
    int mid = lo + (hi - lo) / 2;
    if (condition(mid)) hi = mid;  // can do it, try smaller
    else lo = mid + 1;             // can't, need larger
}

Search in Rotated Array

// LC 33: one half is always sorted
int lo = 0, hi = n - 1;
while (lo <= hi) {
    int mid = lo + (hi - lo) / 2;
    if (nums[mid] == target) return mid;
    if (nums[lo] <= nums[mid]) {  // left half sorted
        if (target >= nums[lo] && target < nums[mid]) hi = mid - 1;
        else lo = mid + 1;
    } else {  // right half sorted
        if (target > nums[mid] && target <= nums[hi]) lo = mid + 1;
        else hi = mid - 1;
    }
}

Monotone Stack

Stack that maintains elements in monotone order. Solves "next greater/smaller element" and related problems in O(n).

Template: Next Greater Element

// For each element, find the next element that is greater
int[] result = new int[n];
Arrays.fill(result, -1);
Deque<Integer> stack = new ArrayDeque<>();  // stores indices

for (int i = 0; i < n; i++) {
    while (!stack.isEmpty() && nums[stack.peek()] < nums[i]) {
        result[stack.pop()] = nums[i];
    }
    stack.push(i);
}

Template: Previous Smaller Element

// For each element, find distance to previous smaller
int[] prevSmaller = new int[n];
Deque<Integer> stack = new ArrayDeque<>();

for (int i = 0; i < n; i++) {
    while (!stack.isEmpty() && nums[stack.peek()] >= nums[i])
        stack.pop();
    prevSmaller[i] = stack.isEmpty() ? -1 : stack.peek();
    stack.push(i);
}

Interval Problems

Sort by start (or end) time, then sweep. Core technique for scheduling and merging problems.

Pattern 1: Merge Intervals

// LC 56 Merge Intervals
Arrays.sort(intervals, (a, b) -> a[0] - b[0]);
List<int[]> merged = new ArrayList<>();
for (int[] interval : intervals) {
    if (merged.isEmpty() || merged.get(merged.size()-1)[1] < interval[0])
        merged.add(interval);
    else
        merged.get(merged.size()-1)[1] =
            Math.max(merged.get(merged.size()-1)[1], interval[1]);
}

Pattern 2: Non-overlapping (Greedy by end time)

// LC 435 Non-overlapping Intervals: min removals
Arrays.sort(intervals, (a, b) -> a[1] - b[1]);  // sort by END
int end = Integer.MIN_VALUE, remove = 0;
for (int[] interval : intervals) {
    if (interval[0] >= end) end = interval[1];  // keep
    else remove++;  // overlaps, remove
}

Pattern 3: Meeting Rooms (sweep line)

// LC 253 Meeting Rooms II: min rooms needed = max concurrent
int[] starts = new int[n], ends = new int[n];
for (int i = 0; i < n; i++) { starts[i] = intervals[i][0]; ends[i] = intervals[i][1]; }
Arrays.sort(starts); Arrays.sort(ends);

int rooms = 0, maxRooms = 0, s = 0, e = 0;
while (s < n) {
    if (starts[s] < ends[e]) { rooms++; s++; }
    else { rooms--; e++; }
    maxRooms = Math.max(maxRooms, rooms);
}

Spiral Order

// LC 54 Spiral Matrix
int top = 0, bottom = m-1, left = 0, right = n-1;
while (top <= bottom && left <= right) {
    for (int j = left; j <= right; j++) result.add(matrix[top][j]);
    top++;
    for (int i = top; i <= bottom; i++) result.add(matrix[i][right]);
    right--;
    if (top <= bottom) {
        for (int j = right; j >= left; j--) result.add(matrix[bottom][j]);
        bottom--;
    }
    if (left <= right) {
        for (int i = bottom; i >= top; i--) result.add(matrix[i][left]);
        left++;
    }
}

String Algorithms

KMP (Knuth-Morris-Pratt): O(n+m) pattern matching

// Build failure function (longest proper prefix-suffix)
int[] buildLPS(String pattern) {
    int m = pattern.length();
    int[] lps = new int[m];
    int len = 0, i = 1;
    while (i < m) {
        if (pattern.charAt(i) == pattern.charAt(len)) {
            lps[i++] = ++len;
        } else if (len > 0) {
            len = lps[len - 1];
        } else {
            lps[i++] = 0;
        }
    }
    return lps;
}

Rabin-Karp: Rolling Hash for pattern matching

// Hash = s[0]*base^(m-1) + s[1]*base^(m-2) + ... + s[m-1]
// Roll: remove leftmost char, add new rightmost char
long hash = 0, patHash = 0, power = 1;
for (int i = 0; i < m; i++) {
    hash = (hash * BASE + s.charAt(i)) % MOD;
    patHash = (patHash * BASE + pattern.charAt(i)) % MOD;
    if (i > 0) power = power * BASE % MOD;
}
// Slide: hash = (hash - s[i]*power) * BASE + s[i+m]

Trie (Prefix Tree)

class TrieNode {
    TrieNode[] children = new TrieNode[26];
    boolean isEnd;
}

void insert(String word) {
    TrieNode node = root;
    for (char c : word.toCharArray()) {
        if (node.children[c - 'a'] == null)
            node.children[c - 'a'] = new TrieNode();
        node = node.children[c - 'a'];
    }
    node.isEnd = true;
}

Palindrome: Expand from Center O(n²)

int expand(String s, int left, int right) {
    while (left >= 0 && right < s.length()
           && s.charAt(left) == s.charAt(right)) {
        left--; right++;
    }
    return right - left - 1;
}
// Call expand(s, i, i) for odd, expand(s, i, i+1) for even

Manacher's Algorithm: O(n) all palindromes

Purpose: Find the longest palindromic substring (or all palindrome radii) in O(n).
Key idea: Reuse previously computed palindrome info. Maintain center C and right boundary R of the rightmost known palindrome.

// Transform: "abc" -> "^#a#b#c#$" (handles odd/even uniformly)
int[] manacher(String s) {
    // Build transformed string
    StringBuilder t = new StringBuilder("^");
    for (char c : s.toCharArray()) t.append('#').append(c);
    t.append("#$");
    int n = t.length();
    int[] p = new int[n];  // p[i] = radius of palindrome at i

    int C = 0, R = 0;  // center, right boundary
    for (int i = 1; i < n - 1; i++) {
        int mirror = 2 * C - i;
        if (i < R) p[i] = Math.min(R - i, p[mirror]);

        // Attempt to expand
        while (t.charAt(i + p[i] + 1) == t.charAt(i - p[i] - 1))
            p[i]++;

        // Update center if expanded past R
        if (i + p[i] > R) { C = i; R = i + p[i]; }
    }
    return p;
    // Longest palindrome length = max(p), start = (maxIdx - maxP) / 2
}

Z-Algorithm: O(n) pattern matching & string analysis

Z[i] = length of the longest substring starting at i that is also a prefix of the string.
Pattern matching: Concatenate pattern + "$" + text, then Z[i] == len(pattern) indicates a match.
Also useful for: longest prefix-suffix, string period, counting distinct substrings.

int[] zFunction(String s) {
    int n = s.length();
    int[] z = new int[n];
    int l = 0, r = 0;  // [l, r] = rightmost Z-box

    for (int i = 1; i < n; i++) {
        if (i < r)
            z[i] = Math.min(r - i, z[i - l]);

        // Try to extend
        while (i + z[i] < n && s.charAt(z[i]) == s.charAt(i + z[i]))
            z[i]++;

        // Update Z-box
        if (i + z[i] > r) { l = i; r = i + z[i]; }
    }
    return z;
}
