Primes & Factorization

Primality Test: O(sqrt(n))

boolean isPrime(int n) {
    if (n < 2) return false;
    for (int i = 2; i * i <= n; i++)
        if (n % i == 0) return false;
    return true;
}

Sieve of Eratosthenes: O(n log log n)

boolean[] sieve = new boolean[n + 1];
Arrays.fill(sieve, true);
sieve[0] = sieve[1] = false;
for (int i = 2; i * i <= n; i++) {
    if (sieve[i]) {
        for (int j = i * i; j <= n; j += i)
            sieve[j] = false;
    }
}

Prime Factorization: O(sqrt(n))

List<int[]> factorize(int n) {
    List<int[]> factors = new ArrayList<>();
    for (int i = 2; i * i <= n; i++) {
        int count = 0;
        while (n % i == 0) { n /= i; count++; }
        if (count > 0) factors.add(new int[]{i, count});
    }
    if (n > 1) factors.add(new int[]{n, 1});
    return factors;
}

Modular Arithmetic

Key Properties

(a + b) % m = ((a % m) + (b % m)) % m
(a * b) % m = ((a % m) * (b % m)) % m
(a - b) % m = ((a % m) - (b % m) + m) % m (avoid negative)
Division requires modular inverse!

Modular Inverse (when mod is prime)

a^(-1) mod p = a^(p-2) mod p    (Fermat's Little Theorem)
static final int MOD = 1_000_000_007;

long modInverse(long a, long mod) {
    return power(a, mod - 2, mod);
}

long power(long base, long exp, long mod) {
    long result = 1;
    base %= mod;
    while (exp > 0) {
        if ((exp & 1) == 1) result = result * base % mod;
        base = base * base % mod;
        exp >>= 1;
    }
    return result;
}

When to Use mod 10^9+7

Signal: "Return the answer modulo 10^9+7" — means the answer can be astronomically large. Apply mod at every addition/multiplication step to prevent overflow.

Combinatorics

Core Formulas

C(n, k) = n! / (k! * (n-k)!)    |    P(n, k) = n! / (n-k)!

Computing C(n, k) mod p

// Precompute factorials and inverse factorials
long[] fact = new long[N];
long[] inv = new long[N];
fact[0] = 1;
for (int i = 1; i < N; i++)
    fact[i] = fact[i-1] * i % MOD;
inv[N-1] = power(fact[N-1], MOD - 2, MOD);
for (int i = N-2; i >= 0; i--)
    inv[i] = inv[i+1] * (i+1) % MOD;

long comb(int n, int k) {
    if (k < 0 || k > n) return 0;
    return fact[n] * inv[k] % MOD * inv[n-k] % MOD;
}

Catalan Numbers

C_n = C(2n, n) / (n+1) = (2n)! / ((n+1)! * n!)

Inclusion-Exclusion Principle

// |A1 ∪ A2 ∪ ... ∪ An| = Σ|Ai| - Σ|Ai∩Aj| + Σ|Ai∩Aj∩Ak| - ...
// Common use: count numbers NOT divisible by any in a set
long count = 0;
for (int mask = 1; mask < (1 << primes.length); mask++) {
    long product = 1;
    int bits = Integer.bitCount(mask);
    for (int i = 0; i < primes.length; i++)
        if ((mask & (1 << i)) != 0) product *= primes[i];
    count += (bits % 2 == 1 ? 1 : -1) * (n / product);
}

GCD / LCM

Euclidean Algorithm: O(log(min(a,b)))

int gcd(int a, int b) {
    return b == 0 ? a : gcd(b, a % b);
}

long lcm(long a, long b) {
    return a / gcd(a, b) * b;  // divide first to avoid overflow
}

Key Properties

gcd(a, b) * lcm(a, b) = a * b
gcd(a, 0) = a
gcd(a, b) = gcd(b, a % b)
gcd(ka, kb) = k * gcd(a, b)
If gcd(a, b) = 1, a and b are coprime

Extended Euclidean (ax + by = gcd(a,b))

// Returns gcd, sets x and y such that ax + by = gcd
int[] extGcd(int a, int b) {
    if (b == 0) return new int[]{a, 1, 0};
    int[] r = extGcd(b, a % b);
    return new int[]{r[0], r[2], r[1] - (a/b) * r[2]};
}

Template: O(log n)

long power(long base, long exp, long mod) {

        if ((exp & 1) == 1)

Computational Geometry

Distance & Cross Product

// Euclidean distance squared (avoid sqrt for comparison)
long dist2(int[] a, int[] b) {
    long dx = a[0] - b[0], dy = a[1] - b[1];
    return dx*dx + dy*dy;
}

// Cross product of vectors OA and OB
// > 0: counter-clockwise, < 0: clockwise, = 0: collinear
long cross(int[] O, int[] A, int[] B) {
    return (long)(A[0]-O[0])*(B[1]-O[1])
         - (long)(A[1]-O[1])*(B[0]-O[0]);
}

Convex Hull (Andrew's Monotone Chain): O(n log n)

// Sort points, build lower then upper hull
Arrays.sort(points, (a, b) -> a[0] != b[0] ? a[0]-b[0] : a[1]-b[1]);
int n = points.length, k = 0;
int[][] hull = new int[2*n][2];

// Lower hull
for (int i = 0; i < n; i++) {
    while (k >= 2 && cross(hull[k-2], hull[k-1], points[i]) <= 0) k--;
    hull[k++] = points[i];
}
// Upper hull
for (int i = n-2, t = k+1; i >= 0; i--) {
    while (k >= t && cross(hull[k-2], hull[k-1], points[i]) <= 0) k--;
    hull[k++] = points[i];
}

Triangle area = |cross(A,B,C)| / 2
Three points collinear iff cross = 0

Common Patterns

// Find the only element appearing once (others appear twice)
int result = 0;
for (int num : nums) result ^= num;

// Find two unique numbers (others appear twice)
int xorAll = 0;
for (int num : nums) xorAll ^= num;
int diffBit = xorAll & (-xorAll);  // lowest differing bit
int a = 0, b = 0;
for (int num : nums) {
    if ((num & diffBit) == 0) a ^= num;
    else b ^= num;
}

Probability & Random

Reservoir Sampling: O(1) space for random selection from stream

// Select k items uniformly from a stream of unknown length
int[] reservoir = new int[k];
for (int i = 0; i < n; i++) {
    if (i < k) {
        reservoir[i] = stream[i];
    } else {
        int j = random.nextInt(i + 1);
        if (j < k) reservoir[j] = stream[i];
    }
}

Josephus Problem

n people in a circle, eliminate every k-th person. Who survives?
Recurrence: J(n, k) = (J(n-1, k) + k) % n, with J(1, k) = 0.
O(n) iterative. For k=2, closed form: J(n) = 2*L + 1 where n = 2^m + L.

// LC 1823: Josephus problem, 0-indexed
int findTheWinner(int n, int k) {
    int survivor = 0;  // base case: 1 person, index 0
    for (int i = 2; i <= n; i++)
        survivor = (survivor + k) % i;
    return survivor + 1;  // convert to 1-indexed
}

Matrix Exponentiation

Solve linear recurrences in O(k^3 * log n) where k is the matrix size.

Template

long[][] multiply(long[][] A, long[][] B, long mod) {
    int n = A.length;
    long[][] C = new long[n][n];
    for (int i = 0; i < n; i++)
        for (int k = 0; k < n; k++)
            if (A[i][k] != 0)
                for (int j = 0; j < n; j++)
                    C[i][j] = (C[i][j] + A[i][k] * B[k][j]) % mod;
    return C;
}

long[][] matPow(long[][] M, long exp, long mod) {
    int n = M.length;
    long[][] result = new long[n][n];
    for (int i = 0; i < n; i++) result[i][i] = 1; // identity
    while (exp > 0) {
        if ((exp & 1) == 1) result = multiply(result, M, mod);
        M = multiply(M, M, mod);
        exp >>= 1;
    }
    return result;
}

Applications

Fibonacci in O(log n): M = [[1,1],[1,0]], M^n gives F(n)
Any linear recurrence: f(n) = a*f(n-1) + b*f(n-2) + ...
Graph paths of length k: (adjacency matrix)^k
State machine transitions: transition matrix ^ steps
